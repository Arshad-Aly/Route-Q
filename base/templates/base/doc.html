RouteQ Documentation

Voice Command Basics

To use the voice search feature, follow these steps:

  1. Click on the microphone icon.
  2. After activation, speak your search query clearly and concisely.
  3. Wait for the system to process your request.

Search Commands

Use the following format for voice search queries:

  • Single location: "Navigate to [Location]"
  • Route search: "Show route from [Location1] to [Location2]"

Example: "Show route from New York to Central Park"

Important: When typing searches, use lowercase for all words except the first letter of location names. Include "from" before the first location and "to" between locations.

Map Interaction

After your location is shown on the map, you can use these features:

  • Voice zoom: Say "Zoom in" or "Zoom out" into the microphone on the right side of the map.
  • Distance measurement: Click the location icon on the right side to find the straight-line distance between two points. Note: This is a straight line distance, not a road route.
  • Layer toggle: Use the layers icon to switch between different map views.

Tips for Best Results

  • Speak clearly and at a moderate pace when using voice commands.
  • Use specific location names for accurate results.
  • If the system doesn't understand, try rephrasing your request.
  • When typing, remember to use the correct format: lowercase with capitalized location names.
